Monolite

A free, premium quality, responsive one page Bootstrap theme based on Grayscale. It also works great as a template for documentation!

About Monolite

Monolite is a premium quality, free Bootstrap 3 theme created based on Grayscale by Start Bootstrap. It can be yours right now, all you need to do is download the template on the preview page. You can use this template for any purpose, personal or commercial.

With this template, just the slightest splash of color can make a huge impact on the overall presentation and design.
